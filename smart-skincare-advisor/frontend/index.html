⚙️ CNN Architecture

A 3-block convolutional network trained end-to-end on dermoscopy images.

Conv2D 3→32 → BatchNorm + ReLU → MaxPool 2×2 → Conv2D 32→64 → BatchNorm + ReLU → MaxPool 2×2 → Conv2D 64→128 → Flatten → FC 512 → FC 128 → Softmax (7)
